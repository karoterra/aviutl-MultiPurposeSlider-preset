# さぶねぼーん氏の立ち絵用

[さぶねぼーん](https://twitter.com/SabuneB)氏の立ち絵用の多目的スライダースクリプトです。

主な機能
- スライダーによる視線切り替え
- 設定ダイアログでの目パチ、口パク調整
